Prework Study Guide

HTML

  • The head element stores important information about the document, such as the title, as well links to CSS and JS files.
  • The body element contains everything to be rendered on the page.

CSS

  • The CSS contains styling information about the contents of the page.
  • It controls the aesthetics of elements on the page.
  • A CSS block starts with the name of the element to be rendered, followed by curly braces, which are then followed by key-value pairs.

JavaScript

  • JavaScript controls the logic of the page and interactive elements.
  • JavaScript is a full programming language.

Git

  • Git is responsible for version control.
